# API RESTAssured CI/CD - Weather Testing

Proyecto de pruebas automatizadas de API utilizando **Java + RestAssured + Maven**, integrado con **GitHub Actions** para ejecución continua (CI/CD).

# Objetivos del proyecto
- Validar respuestas de la API de OpenWeatherMap.
- Practicar automatización de pruebas con RestAssured.
- Integrar pipeline CI/CD con GitHub Actions.

# Tecnologías utilizadas
- **Java 8+**
- **Maven**
- **JUnit 5**
- **RestAssured**
- **GitHub Actions** (CI/CD)
